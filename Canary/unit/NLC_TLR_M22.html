Problemas sobre funciones

Una función es una regla de correspondencia entre dos variables de tal forma que a cada uno de los valores de una de las variables le corresponde un solo valor de la segunda. Las funciones pueden representarse por medio de una ecuación, una tabla o un gráfico.

Las tablas y graficas están íntimamente relacionadas puesto que en una tabla se muestra la relación entre dos variables en una serie de renglones y columnas. Si se considera el par de valores numéricos de cada renglón como las coordenadas de un punto, entonces se obtiene la gráfica correspondiente.
