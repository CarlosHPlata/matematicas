Funciones Trigonométricas

Es indispensable saber la relación entre los lados de un triángulo rectángulo (las tres primeras son suficientes pues las otras tres son sus recíprocos).

Sen A = \frac{\text{Cat. Opuesto}}{\text{Hipotenusa}} = \frac{a}{b}


Cos A = \frac{ \text{ Cat. Adyacente } }{ \text{ Hipotenusa } } = \frac{c}{b}


Tg A = \frac{ \text{ Cat. Opuesto } }{ \text{ Cat. Adyacente } } = \frac{a}{c}
